# Portfolio + Admin (merged)

Single Vite + React + TypeScript app containing both the public portfolio
(`/`) and the admin panel (`/admin`), backed by Supabase.

See **[DEPLOY.md](./DEPLOY.md)** for the full setup and deployment guide
(database migration, creating your admin login, environment variables,
and deploying to Vercel).

```
npm install
npm run dev
```
